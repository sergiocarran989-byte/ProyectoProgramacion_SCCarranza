ProyectoProgramacion_SCCarranza

## Descripción
Proyecto de programación en Java desarrollado en 1 DAW.
Implementa un sistema energético básico con gestión de energía.

## Estructura
- src/Energia.java - Clase que representa una empresa.
- src/Autoconsumo.java - Clase que representa el autoconsumo solar.
- src/Climaticacion.java - Clase que representa el uso de sistemas de climatización.
- src/Electricidad.java - Clase que representa el consumo eléctrico de la empresa.
- src/FuenteEmision.java - Clase abstraca que representa el comportamiento común para todas las fuentes de emisión.
- src/Main.java - Clase principal para ejecutar la simulación de sostenibilidad.
-src/MedidaCorrectora.java - Clase que define el comportamiento de una medida correctora.
-src/MedidaCorrectoraBase - Clase base para las medidas correctoras.
-src/MovilidadElectrica.java - Clase q sustituye vehículos de combustión por eléctricos.
-src/Transporte.java - Clase que representa el consumo de combustible en transporte.
